# DocuMind AI Backend

AI-powered document question answering backend built with:

- Node.js
- Express.js
- LangChain
- LangGraph
- Pinecone
- HuggingFace Embeddings
- Groq API

## Features

- PDF Upload
- PDF Parsing
- Chunking
- Vector Embeddings
- Pinecone Vector Database
- Retrieval-Augmented Generation (RAG)
- General AI Chat
- REST APIs

## APIs

POST /api/document/upload

POST /api/chat

POST /api/rag

POST /api/document/search
